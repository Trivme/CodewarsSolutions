### Return the day
https://www.codewars.com/kata/59dd3ccdded72fc78b000b25
```javascript
function seatsInTheater(nCols, nRows, col, row) {
  let x =  nCols - col + 1;
  let y = nRows - row;
  return x * y;
}
```
